Gerador de Texto Personalizado para o Discord

SOBRE

Este é um site simples que cria mensagens coloridas do Discord usando os códigos de cores ANSI disponíveis nas versões mais recentes da área de trabalho do Discord.

Para usar isso, escreva seu texto, selecione partes dele e atribua cores a elas, depois copie usando o botão abaixo e envie uma mensagem no Discord.

CÓDIGO FONTE

Este aplicativo é executado inteiramente em seu navegador e o código-fonte está disponível gratuitamente em GitHub.
